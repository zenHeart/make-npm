# npm 模板

## 项目结构
* `src|lib` 存放项目源码
* `tests|test` 测试文件
* `docs|doc` 存放项目文档
* `scripts|script` 存放脚本
* `examples|example` 存放项目示例
* `README.md` 项目说明文档
* `package.json` 包文件
* `.*` 各种应用配置例如 `eslint,babelrc,gitignore,npmignore...`

可选的目录文件包括:
* `config` 配置文件夹
* `tools|utils` 存放工具类
* `config` 配置文件夹
* `build` 存放编译脚本
* `bin` cli 命令的入口文件
* `man` cli 命令的 man 手册
* `playground` 实验性质代码(可忽略)
* `CHANGELOG.md` 存放版本迭代信息
* `TODO.md` 存放项目代办事项(可忽略)


## README 模板
